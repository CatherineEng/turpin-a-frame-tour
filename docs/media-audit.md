# Media Audit — Turpin Hubs Archive

## Purpose

This document inventories and contextualizes media files contained in the original Mozilla Hubs export (`hubs-archive/`).  
Its purpose is **mapping and understanding**, not migration.

No files listed here are considered Phase 1 exhibit assets unless and until they are explicitly promoted and documented elsewhere.

---

## Scope

This audit covers media found in:

- `hubs-archive/media/`
- (Referenced by) `hubs-archive/rooms/`

Media types may include:
- audio (`.mp3`, `.wav`)
- images (`.png`, `.jpg`)
- 3D assets (`.glb`)
- other uploaded files

---

## Method

For each media file, record:
- filename (as-is)
- known or suspected function
- evidence for that interpretation
- current status (unassigned / candidate / promoted)

Uncertainty is expected and acceptable.
